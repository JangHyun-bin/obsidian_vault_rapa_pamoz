# 1. 컨소시엄 내 파모즈 역할

## 1.1 컨소시엄 4개 기관 구성

| **기관** | **지위** | **주 역할** |
| --- | --- | --- |
| **㈜레몬헬스케어** | 주관기관 | 사업 총괄 운영, 800만 가입자 의료앱 운영, EMR 연동 API(QAB), 환자 모집·실증 운영, 카카오 알림톡, KPI 측정 |
| **㈜파모즈** | 참여기관 1 | **측위·AR 시스템 기술 구현 주체. 현장 인프라 구축 주체** |
| **전남대학교산학협력단** | 참여기관 2 | 의료 도메인 RAG·온톨로지, Safety Guardrail, 의학 자문, IRB·학술 성과, 평가셋 검증 |
| **전남대학교병원** | 수요기관 | 실증 환경 제공, EMR·HIS 운영, 환자 모집 협조, 보안 심의, 현장 작업 허가 |

## 1.2 파모즈 담당 역할 요약

**측위·AR 시스템의 기술 구현 주체이자 병원 현장 물리 인프라 구축 주체.**

컨소시엄 내 가장 복잡한 기술·현장 협의를 주관하며, RAPA에 보고되는 핵심 기술 산출물의 1차 책임을 담당.

## 1.3 파모즈 작업 카테고리

| **영역** | **내용** |
| --- | --- |
| **기술 구현** | 측위 백엔드·SDK / AR 모듈 |
| **현장 인프라 구축** | LiDAR 스캔 / 지자기 핑거프린팅 DB 구축 / QR 앵커 50~100개소 / BLE 5.1 AoA 10~20개소 / 측위 테스트 |
